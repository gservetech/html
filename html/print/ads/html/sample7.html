Shape any element with clip-path

Circles, ovals, polygons, insets, or fully custom path shapes — all with one property. Each demo includes a small code snippet and a "Show code" toggle.

Circle

Perfect for avatars or profile photos. Fallback: border-radius.

<!-- CSS -->
.circle .media {
  border-radius: 50%; /* fallback */
  clip-path: circle(50% at 50% 50%);
}

Ellipse / Oval

Great for banners and soft edges.

<!-- CSS -->
.oval .media {
  border-radius: 999px; /* friendly fallback */
  clip-path: ellipse(80% 45% at 50% 50%);
}

Inset

Cuts into the edges — think reverse padding.

<!-- CSS -->
.inset .media {
  clip-path: inset(12% 8% 16% 8% round 14px);
}

Polygon (trapezoid)

Slanted cards, angled buttons, or decorative sections.

<!-- CSS -->
/* trapezoid */
.polygon .media {
  clip-path: polygon(0 0, 100% 0, 85% 100%, 20% 100%);
}

/* alternate: parallelogram */
.polygon.alt .media {
  clip-path: polygon(10% 0, 100% 0, 90% 100%, 0 100%);
}

Path (custom curve)

Absolute control: curves, arcs, and brand shapes using path()

<!-- CSS -->
/* heart-like shape (example, path can be any valid SVG path) */
.path .media {
  clip-path: path('M10 30 A20 20 0 0 1 50 30 A20 20 0 0 1 90 30 Q90 60 50 90 Q10 60 10 30 Z');
}

/* alternate path (rounded blob) */
.path.blob .media {
  clip-path: path('M10 50 C 10 20, 40 0, 60 20 C 80 40, 70 80, 40 90 C 10 95, 10 70, 10 50 Z');
}

Notes & fallbacks

clip-path support is broad, but path() & advanced shapes vary by browser. Provide a fallback (border-radius, background, or a plain container). Use feature queries when needed.

<!-- Example feature query -->
@supports (clip-path: circle(50%)) {
  .avatar { clip-path: circle(50% at 50% 50%); }
}
